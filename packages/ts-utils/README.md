An NPM package that can be added to a Node.js/NPM codebase as follows:

```shell
$ npm add @lionweb/ts-utils
```

It contains general TypeScript utilities, e.g. for working with maps and such.


## Development

Build it from source as follows:

```shell
npm run build
```
